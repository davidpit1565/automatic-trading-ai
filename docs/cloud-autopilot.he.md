# רובוט בענן + התראות טלגרם (רקע אמיתי)

הרובוט רץ בענן של GitHub כל ~5 דקות — לבד, בלי מחשב, בלי שהאתר יהיה פתוח —
ושולח לך הודעת טלגרם על כל קנייה/מכירה. **כסף מדומה בלבד.**

## הגדרה חד-פעמית (~5 דקות, מהטלפון)

### 1. יצירת בוט טלגרם
1. פתח טלגרם, חפש **@BotFather**, לחץ Start.
2. שלח `/newbot`, תן שם כלשהו — תקבל **טוקן** (מחרוזת ארוכה כמו
   `123456:ABC-DEF...`). העתק אותו.

### 2. קבלת ה-Chat ID שלך
1. שלח הודעה כלשהי לבוט החדש שיצרת (כל דבר, "hi").
2. פתח בדפדפן: `https://api.telegram.org/bot<הטוקן>/getUpdates`
   (הדבק את הטוקן במקום `<הטוקן>`).
3. חפש בטקסט `"chat":{"id":` — המספר אחריו הוא ה-**Chat ID**.

### 3. הזנת הקודים ב-GitHub (פעם אחת)
1. פתח: `github.com/davidpit1565/automatic-trading-ai/settings/secrets/actions`
2. **New repository secret** → שם: `TELEGRAM_BOT_TOKEN` → ערך: הטוקן → Save.
3. שוב **New repository secret** → שם: `TELEGRAM_CHAT_ID` → ערך: ה-Chat ID → Save.

זהו. מהרגע הזה הרובוט רץ בענן כל ~5 דקות ושולח לך התראה על כל עסקה.

## איך מפעילים ידנית עכשיו (בלי לחכות)
`github.com/davidpit1565/automatic-trading-ai/actions` → **Cloud Paper
Autopilot** → **Run workflow**.

## אילו הודעות מגיעות אליך (הכול בעברית)
- **על כל קנייה/מכירה** — התראה מיידית (🟢 קנייה / 🔴 מכירה), רק על העסקה של
  אותו רגע.
- **פעם ביום בערב** — סיכום כללי: שווי תיק, מזומן, רווח/הפסד, ופוזיציות פתוחות.
  נשלח בסביבות **22:00** לפי אזור הזמן שמוגדר (ברירת מחדל: שעון ישראל).
- אין ספאם: בין לבין הרובוט רץ בשקט ולא שולח כלום.

## שינוי אזור הזמן של הסיכום היומי (כשנוסעים)
כדי שהסיכום יגיע ב-22:00 במקום שבו אתה נמצא, הגדר משתנה ב-GitHub:
`.../settings/variables/actions` → **New repository variable** → שם:
`SUMMARY_TIMEZONE` → ערך: שם אזור הזמן, למשל `Europe/Brussels` (בלגיה) או
`Asia/Jerusalem` (ישראל). מעבר קיץ/חורף מטופל אוטומטית.

## בדיקת התראות ידנית
ב-`.../actions` → **Cloud Paper Autopilot** → **Run workflow**, סמן את
**"Send a Telegram test message"** → הרובוט ישלח לך הודעת בדיקה מיד.

## מה שחשוב לדעת (ביושר)
- **כסף מדומה** — הרובוט לא נוגע בכסף אמיתי ולא יכול. הבוט רק קורא מחירים
  ושולח הודעות.
- רץ כל ~5 דקות (תזמון GitHub הוא "מאמץ מיטבי" — לפעמים באיחור).
- בשוק יורד הוא לרוב לא יקנה — וזה בכוונה (הגנה על ההון).
- התזמון מושבת אוטומטית ע"י GitHub אחרי 60 יום ללא פעילות במאגר; אם זה קורה,
  היכנס ל-Actions והפעל שוב.
- אם לא הגדרת טלגרם — הרובוט עדיין רץ וצובר היסטוריה, פשוט בלי התראות.
