## שלב 1 — התקנות חד-פעמיות (~10 דקות)

1. **Node.js** — הורד והתקן מ- <https://nodejs.org> (גרסת LTS, 20 ומעלה).
   ב-Windows: הורד את קובץ ההתקנה והתקן כרגיל. ב-Mac: אותו דבר.
2. פתח **טרמינל** (ב-Mac: אפליקציית Terminal; ב-Windows: PowerShell).
3. הורד את הפרויקט:

   ```bash
   git clone -b claude/market-scan-integration-r9lck1 https://github.com/davidpit1565/automatic-trading-ai.git
   cd automatic-trading-ai
   npm install
   ```

   (אם שמרת את קובץ ה-ZIP ב-iCloud — אפשר במקום זה לחלץ אותו לתיקייה,
   להיכנס אליה בטרמינל עם `cd`, ולהריץ `npm install`.)

## שלב 2 — בדיקה ראשונה במצב דמו (דקה)

```bash
npm start
```

פתח בדפדפן: <http://localhost:5173> — הדשבורד יעלה עם באנר צהוב
("DEMO data"). זה תקין: עוד אין מפתח לנתונים אמיתיים.

עצירה: `Ctrl+C` בטרמינל.

## שלב 3 — חיבור לנתונים אמיתיים מ-Revolut X (~10 דקות)

הכול **לקריאה בלבד** — המפתח לא יכול לגעת בכסף שלך.

1. צור צמד מפתחות הצפנה על המחשב שלך (בטרמינל, בתוך תיקיית הפרויקט):

   ```bash
   openssl genpkey -algorithm ed25519 -out revx-private.pem
   openssl pkey -in revx-private.pem -pubout -out revx-public.pem
   ```

2. פתח את **Revolut X** בדפדפן → הגדרות → **API keys** → צור מפתח חדש:
   - הדבק את התוכן של `revx-public.pem` (פתח אותו בעורך טקסט והעתק הכול).
   - בחר הרשאות **קריאה בלבד** (Read only). לא לסמן מסחר!
   - שמור את מזהה ה-API key שתקבל.

3. צור קובץ הגדרות:

   ```bash
   cp .env.example .env
   ```

   פתח את `.env` בעורך טקסט ומלא:

   ```
   REVX_API_KEY=המזהה-שקיבלת
   REVX_PRIVATE_KEY_PATH=./revx-private.pem
   ```

4. הפעל שוב `npm start` ורענן את הדפדפן — הבאנר יהפוך **ירוק**:
   נתונים חיים.

> 🔒 הקבצים `.env` ו-`*.pem` לעולם לא נשמרים ב-Git ולא עוזבים את המחשב.

## שלב 4 — הפעלת הטייס האוטומטי (דקה)

1. בדשבורד: טאב **Portfolio**.
2. תחת **Paper Autopilot**: בחר מחזור (מומלץ `1h`) ולחץ **Start autopilot**.
3. זהו. מעכשיו הוא סורק, נכנס ויוצא לבד — בכסף מדומה — כל עוד הדפדפן פתוח.
   אחרי רענון הדף הוא ממשיך לבד (אלא אם לחצת על מתג החירום).

מתג חירום: כפתור **⛔ Kill switch** עוצר הכול מיד.

## שלב 5 — השגרה שלך

- פעמיים-שלוש בשבוע: פתח את טאב Portfolio.
- הסתכל על **Performance feedback**: האם רמות ביטחון גבוהות באמת מנצחות?
  האם המערכת מכה את "פשוט להחזיק"? כמה עסקאות נסגרו בסטופ מול יעד?
- פעם בשבוע: לחץ **Download backup** ושמור את הקובץ (למשל ב-iCloud) —
  זה כל תיעוד הביצועים שלך.
